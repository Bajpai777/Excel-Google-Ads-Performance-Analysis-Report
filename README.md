# Google Ads Performance Analysis Report
This Excel report presents a detailed overview of Google Ads campaign performance. It summarizes critical advertising metrics including total impressions, clicks, CTR, conversions, conversion rate, and ROAS to evaluate the overall effectiveness of the marketing spend. The dashboard also highlights cost-efficiency metrics such as CPC and CPA. Visual representations of conversion values and campaign clicks by ad group provide deeper insights into which campaigns are performing well and contributing most to business outcomes. This report supports data-driven decision-making for optimizing campaign strategy and improving return on investment.

## Sample Reports:
1) Impressions.
2) Clicks.
3) Click-Through Rate (CTR) = Clicks / Impressions.
4) Conversions.
5) Conversion Rate = Conversions / Clicks.
6) Total Cost.
7) Cost per Click (CPC) = Cost / Clicks.
8) Cost per Conversion (CPA) = Cost / Conversions.
9) Return on Ad Spend (ROAS) = Conversion Value / Cost.
10) Conversion Value.
11) Campaign CPM.

## Process
- Import file 'Excel Dataset' with use of Power Query.

- Create different Reports, Use maximum features offered in power pivot.
- Use conditional formatting and Different types of appropriate Graphs/Charts Use Slicers, Timeline, Formatting techniques to create a Dashboard.
- Analyze the Dataset and Create a Dashboard.
